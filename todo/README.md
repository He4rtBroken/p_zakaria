# Projet : hello.sh

## Description

Le fichier `hello.sh` est un script simple qui efface l'écran du terminal et affiche un "Hello World" personnaliser.

## Comment utiliser

1. Ouvrir un terminal.
2. Naviguer vers le dossier contenant le fichier `hello.sh`.
    
    ```bash
    cd p_zakaria/todo/
    ```

3. Exécuter le script avec la commande suivante :

    ```bash
    ./hello.sh
    ```

## Erreur de permission
N'oubliez pas de modifier les permissions si cela ne fonctionne pas:
    
```bash
chmod +x hello.sh
```

## Affichage

Vous devrez avoir ce resultat :

```
Hello World Zakaria Garaoui
```
